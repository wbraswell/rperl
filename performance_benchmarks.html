Performance Benchmarks

Last Update: November 10th, 2013

All current benchmarks are based on the manually-compiled bubblesort algorithm, and include all 3 execution modes: pure Perl, RPerl using Perl data types & C/C++ operations, and RPerl using C/C++ data types & C/C++ operations. There are dramatic speed differences between these 3 execution modes. Future testing algorithms will include the Alioth Shootout Benchmark Games, etc.
